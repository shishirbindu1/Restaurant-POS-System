# 🍽️ Restaurant POS System

## Overview

The Restaurant POS (Point of Sale) System is a web application built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. It allows restaurant staff to manage customer orders efficiently by selecting tables, adding menu items, updating quantities, calculating bills, and processing payments.

This project was built to strengthen core JavaScript concepts by simulating a real-world restaurant workflow.

---

## Features

* 🍽️ Table-wise order management
* 📋 Dynamic menu generation
* ➕ Add menu items to an order
* 🔼 Increase item quantity
* 🔽 Decrease item quantity
* ❌ Automatically remove items when quantity reaches zero
* 💰 Automatic item total calculation
* 🧾 Grand total calculation
* 💳 Pay and clear current table order
* 🟢 Occupied and available table status
* ⚡ Dynamic UI updates without page reload
* 📱 Responsive interface built with Tailwind CSS

## How It Works

1. Select a restaurant table.
2. Browse the food menu.
3. Add menu items to the selected table.
4. Increase or decrease item quantities.
5. Automatically calculate item totals and the grand total.
6. Process payment to clear the current table.
7. Track occupied tables using visual indicators.
